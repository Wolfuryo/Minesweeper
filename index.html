Minesweeper rules
Objective
The objective of the game is revealing all the squares that are not mines. Trying to reveal a square that is a mine will result in a loss. The first square to be revealed will never be a mine.
Types of squares
A square can either contain a mine or a number. If it contains a mine and it is clicked on, the game will end in a loss. If it contains a number and it is clicked on, the number will be shown on the screen. The number represents how many mines are there in the adjacent squares. If the square would contain the number 0, then, recursively, all the empty squares around it will be opened.
Clicking a square
Clicking a square will reveal it according to the type that it has.
